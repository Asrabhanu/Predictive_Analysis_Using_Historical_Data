# House Price Prediction Using Linear Regression

## Project Overview
This project predicts house prices using the Linear Regression algorithm. It uses historical housing data to train a machine learning model and predict house prices based on different features.

## Dataset
- Dataset: Housing.csv
- Number of Records: 545
- Features:
  - Area
  - Bedrooms
  - Bathrooms
  - Stories
  - Main Road
  - Guest Room
  - Basement
  - Hot Water Heating
  - Air Conditioning
  - Parking
  - Preferred Area
  - Furnishing Status

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Algorithm
- Linear Regression

## Project Steps
1. Import required libraries.
2. Load the dataset.
3. Preprocess the data.
4. Convert categorical variables into numerical values.
5. Split the dataset into training and testing sets.
6. Train the Linear Regression model.
7. Predict house prices.
8. Evaluate the model using MAE, MSE, and R² Score.
9. Visualize Actual vs Predicted House Prices.

## Results
The model successfully predicts house prices using historical data and visualizes the relationship between actual and predicted values.
